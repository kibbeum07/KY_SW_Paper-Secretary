# KY_SW_Paper-Secretary

# 종이 비서 (Paper Assistant)

AI 기반 문서 해석 및 디지털 자동 입력 서비스

## 프로젝트 소개

복잡한 국가 장학금 신청서, 공공기관 서류, 병원 진료비 내역서 등
종이 문서를 디지털로 옮기는 과정은 많은 사람들에게 어려움을 줍니다.

특히 고령층, 디지털 취약계층, 행정 절차가 익숙하지 않은 대학생들은
복잡한 행정 용어와 신청 절차 때문에 필요한 혜택을 놓치는 경우가 많습니다.

"종이 비서"는 문서를 사진으로 촬영하면
OCR 기술을 통해 텍스트를 추출하고,
어려운 내용을 쉽게 설명하며,
사용자가 다음 행동(신청/제출/결제)까지 바로 진행할 수 있도록 돕는 서비스입니다.

## 해결하고자 하는 사회 문제

- 복잡한 행정 절차로 인한 정보 접근 격차
- 디지털 취약계층의 공공 서비스 이용 어려움
- 어려운 행정 용어로 인한 혜택 누락
- 종이 문서를 직접 입력해야 하는 불편함
- 고령층 및 대학생의 디지털 행정 스트레스 증가

- ## 핵심 기능

### 1. OCR 기반 문서 인식
문서를 촬영하면 텍스트를 자동 추출합니다.

### 2. 문서 유형 자동 분류
- 계약서
- 영수증
- 신청서
- 기타 문서

### 3. 핵심 정보 자동 추출
- 이름
- 주소
- 금액
- 날짜

### 4. 위험 요소 분석
계약서 내 위험 단어 및 만료 여부를 분석합니다.

### 5. 쉬운 말 해석 기능
어려운 행정 용어를 쉬운 언어로 설명합니다.

### 6. 다음 행동 가이드 제공
사용자가 필요한 사이트로 바로 이동할 수 있도록 연결합니다.

## 시스템 동작 흐름

1. 사용자가 문서를 촬영
2. OCR을 통해 텍스트 추출
3. AI가 문서 유형 분류
4. 핵심 정보 자동 추출
5. 위험 요소 분석
6. 쉬운 설명 제공
7. 관련 사이트 연결

## 현재 구현된 기능

### OCR 처리
`extract_text()`
- 문서 이미지에서 텍스트 추출

### 문서 분류
`classify_document()`
- 계약서 / 영수증 / 신청서 자동 분류

### 핵심 정보 추출
`extract_key_info()`
- 이름
- 주소
- 금액
- 날짜 추출

### 위험 분석
`analyze_risk()`
- 위험 키워드 탐지
- 만료일 확인
- 누락 정보 검사

- ## 차별점

기존 OCR 서비스는 단순히 문서를 읽어주는 기능에 집중되어 있습니다.

"종이 비서"는 단순 인식을 넘어,
사용자가 실제로 무엇을 해야 하는지까지 안내합니다.

예시:
- 국가장학금 신청 페이지 연결
- 병원비 납부 사이트 연결
- 계약 위험 요소 경고
- 행정 용어 쉬운 설명 제공

- ## 기존 서비스와의 차이점

| 서비스 | 특징 | 한계 |
|---|---|---|
| 정부 AI 행정지원 | 공무원 업무 보조 | 일반인 접근 어려움 |
| 기업용 OCR 솔루션 | 높은 정확도 | 비용 높음 |
| 시니어 교육 앱 | 사용법 교육 중심 | 즉시 문제 해결 어려움 |
| 법률 계약 분석 앱 | 계약서 분석 특화 | 생활 행정 범용성 부족 |
| 종이 비서 | 생활형 행정 도우미 | 개인 맞춤형 서비스 목표 |
